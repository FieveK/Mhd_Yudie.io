Materi

Di sini Anda akan belajar dasar-dasar pemrograman Python. Mari kita mulai dengan dasar-dasar sintaks, variabel, tipe data, struktur kondisional, looping, fungsi, dan kelas.

Sintaks Dasar

Python menggunakan indentasi untuk menandakan blok kode. Berikut contoh sederhana:

if 5 > 2:
    print("Five is greater than two!")

Variabel

Variabel digunakan untuk menyimpan nilai. Contoh:

x = 5
y = "Hello, World!"

Tipe Data

Python memiliki beberapa tipe data bawaan, seperti angka, string, dan list. Contoh:

x = 5       # int
y = "Hello"  # str
z = [1, 2, 3]  # list

Struktur Kondisional

Struktur kondisional digunakan untuk membuat keputusan dalam kode. Contoh:

if x > y:
    print("x is greater than y")
elif x == y:
    print("x is equal to y")
else:
    print("x is less than y")

Looping

Python memiliki beberapa cara untuk melakukan looping melalui data, seperti menggunakan for dan while loop. Contoh:

for i in range(5):
    print(i)

i = 0
while i < 5:
    print(i)
    i += 1

Fungsi

Fungsi adalah blok kode yang hanya berjalan ketika dipanggil. Contoh:

def my_function():
    print("Hello from a function")

my_function()

Kelas dan Objek

Python adalah bahasa pemrograman berorientasi objek. Hampir semuanya adalah objek, dengan propertinya dan metodenya. Contoh:

class MyClass:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        print(f"Hello, my name is {self.name} and I am {self.age} years old.")

p1 = MyClass("John", 36)
p1.greet()

File I/O

Python dapat digunakan untuk membaca dan menulis file. Contoh:

# Menulis ke file
with open("example.txt", "w") as file:
    file.write("Hello, World!")

# Membaca dari file
with open("example.txt", "r") as file:
    content = file.read()
    print(content)

Exception Handling

Python memiliki mekanisme untuk menangani kesalahan dengan menggunakan blok try-except. Contoh:

try:
    print(x)
except NameError:
    print("Variable x is not defined")
except:
    print("Something else went wrong")

Library Standar

Python memiliki banyak library standar yang bisa digunakan untuk berbagai keperluan. Contoh:

import math

print(math.sqrt(16))
print(math.pi)

Penggunaan Library Populer

Selain library standar, Python juga memiliki banyak library populer seperti NumPy dan Pandas untuk analisis data. Contoh:

import numpy as np

arr = np.array([1, 2, 3, 4, 5])
print(arr)
